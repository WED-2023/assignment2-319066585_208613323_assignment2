About

Submitted by: Yehezkel Kadosi, Tamir Cohen

No template or external plugins such as jQuery were used in the development of this project. All HTML, CSS, and JavaScript were written manually using vanilla JavaScript.

During the development of the project, we encountered several challenges:

1. Implementing the logic for navigating between screens was complex. For example, when switching from the game screen to the home screen, the home screen appeared visually, but the game continued running and playing sounds in the background.

2. After completing one game, clicking the "New Game" button caused an issue where enemy spaceships no longer fired bullets.

3. Ensuring that all fields in the registration form met the required validation criteria.

4. During gameplay, we experienced an issue where moving the player's spaceship using the up/down arrow keys caused the entire webpage to scroll.

5. Implementing accurate collision detection between bullets and enemy spaceships.
